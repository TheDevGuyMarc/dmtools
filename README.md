# DM Tools

The DM Tools are a web application that help dungeon masters to manager their games and worlds digitaly.

## Badges

## Features

- Authentication
- World Management (CMS Style)
- Character Creation
- NPC Creation
  - Generator
- Campaign Creation
- Session Creation
  - Dice Roller
    - Die Roll (d4, d6, d8, d10, d12, d20)
    - Combat Roll
    - Ability Roll
    - Saving Roll
    - Spell Roll
  - Simulations
    - Combat
    - Ability Checks
    - Save Checks
    - Spell Checks
  - Environment Controls (Time + Weather)
  - Session Logs
  - Notes
  - Player Management
    - Hit Points
    - Experience
    - Rewards
    - Ability Points
    - Initiative
    - Inventories
- Plot Creation
  - Encounter Creation
  - Encounter Generator
  - Quest Creation
  - Quest Generator
  - Location Creation
  - Location Generator
  - Shop Creation
  - Shop Generator
- Item Creation
  - Item Generation

**Client:**

- Base:
  - React
  - Axios
  - React-Router
  - Styled Components
  - SCSS
  - Typescript
  - React-OIDC-Context
  - OIDC-Client-TS
- Utilities:
  - Lodash
- Testing:
  - Jest
  - Cypress
  - Mockito
  - Faker

**Server:**

- Base:
  - Java 20
  - Spring Boot 2.7.4
  - Spring Web
  - Spring Security
  - Hibernate
  - Spring Data JPA
  - MariaDB JDBC
- Utilities:
  - Lombok
- Testing:
  - JUnit
  - Mockito
  - Gatling
